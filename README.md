# `erl_ext`

## Description

Set of extensions for *Erlang's* Standard Library, *OTP* and *ERTS* that we use on a daily basis.

## Development

### Building

```bash
$ rebar3 compile
```

### Testing

```bash
$ rebar3 dialyzer
$ rebar3 proper
```
